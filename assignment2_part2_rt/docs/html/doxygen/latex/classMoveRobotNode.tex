\doxysubsection{Detailed Description}
A R\+O\+S2 node class that handles robot movement commands. 

This class creates a publisher to send Twist messages to the cmd\+\_\+vel topic, controlling the robot\textquotesingle{}s linear and angular velocities at regular intervals.
